# Tabor-College-App
### Setting up the development environment
1. On your git bash window, run `git clone https://github.com/Falzaro/Tabor-College-App.git` to clone the Tabor-College-App repository from Github.
2. Then run `cd tabor-college-app` to go to the project directory.
3. To view the app using expo, execute `npm start`.
